# mall-swarm 核心技术栈与关键文件清单

## 📚 核心技术栈

### 1. 微服务框架
- **Spring Cloud 2023.0.1** - 微服务框架
- **Spring Cloud Alibaba 2023.0.1.0** - 阿里巴巴微服务组件
- **Spring Boot 3.2.2** - 应用框架
- **Java 17** - 开发语言

### 2. 服务注册与配置中心
- **Nacos** - 服务注册发现、配置中心
  - 服务发现：`spring-cloud-starter-alibaba-nacos-discovery`
  - 配置管理：`spring-cloud-starter-alibaba-nacos-config`

### 3. API网关
- **Spring Cloud Gateway** - 响应式API网关
  - 路由转发
  - 统一鉴权
  - 跨域处理

### 4. 认证授权
- **Sa-Token 1.37.0** - 轻量级权限认证框架
  - JWT集成
  - Redis存储
  - 多端登录支持（管理员/会员）

### 5. 数据持久化
- **MyBatis 3.5.14** - ORM框架
- **MyBatis Generator 1.4.2** - 代码生成器
- **PageHelper 6.1.0** - 分页插件
- **Druid 1.2.9** - 数据库连接池
- **MySQL 8.0.29** - 关系型数据库

### 6. 缓存与消息队列
- **Redis** - 分布式缓存（Spring Data Redis）
- **RabbitMQ** - 消息队列（订单超时取消等）
- **MongoDB** - NoSQL数据库（会员浏览历史等）

### 7. 搜索引擎
- **Elasticsearch 7.17.3** - 商品搜索
- **Spring Data Elasticsearch** - ES集成

### 8. 对象存储
- **阿里云OSS** - 对象存储服务
- **MinIO** - 开源对象存储

### 9. 支付集成
- **支付宝SDK 4.38.61** - 支付宝支付

### 10. 监控与文档
- **Spring Boot Admin 3.2.2** - 应用监控
- **Knife4j 4.5.0** - API文档（Swagger增强版）
- **Actuator** - 健康检查

### 11. 日志收集
- **Logstash** - 日志收集与处理
- **ELK Stack** - 日志分析

### 12. 容器化部署
- **Docker** - 容器化
- **Kubernetes** - 容器编排
- **Docker Maven Plugin** - 镜像构建

### 13. 工具库
- **Hutool 5.8.16** - Java工具类库

## 💡 关键技术点总结

### 1. 微服务架构模式
- 服务注册发现：Nacos
- 配置中心：Nacos Config
- API网关：Spring Cloud Gateway
- 服务调用：OpenFeign（已移除，使用RestTemplate或WebClient）

### 2. 认证授权方案
- 框架：Sa-Token（替代Spring Security OAuth2）
- Token类型：JWT
- 存储：Redis
- 多端支持：管理员端 + 会员端

### 3. 数据持久化方案
- 关系型数据库：MySQL + MyBatis
- NoSQL：MongoDB（浏览历史、收藏等）
- 缓存：Redis
- 搜索：Elasticsearch

### 4. 消息队列应用
- 技术：RabbitMQ
- 场景：订单超时自动取消（延迟队列 + 死信队列）

### 5. 分布式事务
- 技术：Seata（配置中，实际使用中可能采用最终一致性）

### 6. 监控与运维
- 应用监控：Spring Boot Admin
- 日志收集：ELK Stack
- 容器化：Docker + Kubernetes
